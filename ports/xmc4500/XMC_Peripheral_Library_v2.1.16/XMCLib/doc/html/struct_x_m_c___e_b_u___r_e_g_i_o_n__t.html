Detailed Description

EBU region initialization with read and write configurations

The structure presents a convenient way to set/obtain the read, read timing, write and write timing configurations for a region of EBU. The XMC_EBU_ConfigureRegion() can be used to populate the structure with the region read and write configuration values of EBU.
